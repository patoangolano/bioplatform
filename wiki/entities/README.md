# wiki/entities/ — Entidades

Páginas de referência para ferramentas, datasets, instituições, organismos, plataformas e repositórios.

## Propósito

Documentar entidades recorrentes com:
- Nome canônico e aliases
- Tipo (ferramenta, dataset, organismo, instituição, plataforma)
- Versão ou release relevante
- URL ou identificador persistente
- Relação com o projeto bioplatform
- Páginas wiki que referenciam esta entidade
