# Career Hub

Hub de estrategia profissional. A bioplatform nao e apenas um projeto tecnico -- e um veiculo de posicionamento no mercado de bioinformatica. Este hub conecta execucao tecnica a narrativa de carreira.

## Estrutura do Hub

### Posicionamento

Definicao do perfil profissional: bioinformata com dominio completo do ciclo -- da infraestrutura (Docker, VPS, CI/CD) a analise cientifica (RNA-Seq, paleogenomica). Diferencial: capacidade de operar em todas as 5 camadas do modelo, nao apenas em uma especialidade isolada.

Pontos fortes a desenvolver e evidenciar:
- Pipeline engineering com Nextflow e nf-core
- Containerizacao de ferramentas bioinformaticas
- Infraestrutura como codigo para reproducibilidade
- Analise de dados genomicos com rigor estatistico
- Documentacao e comunicacao cientifica

### Logica de Portfolio

O portfolio nao e uma colecao de scripts soltos. E a propria bioplatform como prova viva de competencia:

1. **Codigo** -- GitHub com CI/CD, testes, documentacao, Dockerfiles
2. **Infraestrutura** -- VPS em producao, servicos reais rodando
3. **Conhecimento** -- Este wiki como evidencia de profundidade conceitual
4. **Processo** -- Historico de commits, issues, decisoes arquiteturais documentadas

### Traducao Projeto-Carreira

Como cada elemento do projeto se traduz em narrativa profissional:

- Pipeline RNA-Seq → "Implementei analise transcriptomica ponta a ponta com Nextflow e Docker"
- Infraestrutura MCP → "Projetei ecossistema de 15 servidores bioinformaticos via MCP"
- Wiki → "Mantenho documentacao cientifica rigorosa como pratica diaria"
- VPS Deploy → "Opero infraestrutura em producao com CI/CD automatizado"
