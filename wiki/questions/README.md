# wiki/questions/ — Perguntas Duráveis

Perguntas preservadas com respostas sintetizadas a partir do wiki.

## Propósito

Quando uma consulta produz uma resposta útil e bem fundamentada, promovê-la para uma página durável aqui. Cada página contém:
- A pergunta original
- A resposta sintetizada
- Referências a páginas wiki e fontes raw
- Data da resposta e possíveis atualizações futuras

## Template

```markdown
***
title: string (a pergunta)
slug: string
type: question
status: seed | draft | active | reviewed
created: YYYY-MM-DD
updated: YYYY-MM-DD
tags: [array]
source_count: integer
source_files: [array]
related_pages: [array]
***

# Pergunta

...

## Resposta

...

## Referências

- [[pagina-1]]
- [[pagina-2]]
```
