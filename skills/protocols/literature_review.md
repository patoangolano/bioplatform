# Protocolo: Revisão de Literatura

## Estratégia de Busca

### PubMed — Termos Obrigatórios
1. Nome da proteína/gene + organismo
2. Função putativa + "structure OR function"
3. Doença associada (se aplicável)

### Filtros Padrão
- Últimos 5 anos (prioridade)
- Artigos de revisão para contexto
- Artigos originais para dados experimentais

## Avaliação de Relevância

| Prioridade | Critério |
|------------|----------|
| Alta | Dados experimentais diretos sobre a proteína-alvo |
| Média | Estudos com homólogos próximos (identity >50%) |
| Baixa | Estudos com proteínas da mesma família |

## Extração de Informação

Para cada artigo relevante, extrair:
- Achados principais (1-2 frases)
- Método utilizado
- Organismo do estudo
- Relevância para a query

## Proveniência
- Busca PubMed → classificação: "observação"
- Extrapolação de homólogos → classificação: "inferência"
- Predição baseada em analogia → classificação: "hipótese"
