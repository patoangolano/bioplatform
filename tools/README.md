# tools/ — Ferramentas Auxiliares

Ferramentas leves de suporte ao repositório.

## Ferramentas futuras

- **Validador de frontmatter** — verifica YAML frontmatter em todas as páginas wiki
- **Validador de links** — detecta links quebrados entre páginas wiki
- **Detector de páginas órfãs** — encontra páginas sem incoming links
- **Detector de páginas obsoletas** — identifica conteúdo desatualizado
- **Exportador de wiki** — gera HTML/PDF a partir das páginas markdown
- **Utilitários grep** — busca rápida em todo o repositório

## Princípios

- Scripts simples, auto-contidos
- Preferência por bash/Python
- Sem dependências pesadas
- Documentados inline
