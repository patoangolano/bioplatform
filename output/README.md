# output/ — Artefatos Gerados

Diretório de trabalho para artefatos antes da promoção para a camada de output da wiki.

## Fluxo

1. Gerar artefato em `output/` (script, análise, figura)
2. Revisar e refinar
3. Promover para `wiki/outputs/reports/`, `wiki/outputs/slides/` ou `wiki/outputs/figures/`

## Organização sugerida

- `output/figures/` — figuras geradas por análises
- `output/tables/` — tabelas exportadas
- `output/reports/` — rascunhos de relatórios
- `output/tmp/` — arquivos temporários de processamento
